# Auto Update 模块功能说明

## 模块概述
自更新模块负责处理应用的自动更新功能，包括版本检查、文件下载、完整性校验、备份管理和更新执行等核心功能。

## 文件功能说明

### 1. `__init__.py` - 模块初始化文件
- **功能**：模块包初始化定义
- **作用**：
  - 定义模块的公共接口和导出类
  - 提供模块级别的导入控制
  - 声明 `AutoUpdater` 类为模块的主要导出对象
- **特性**：简洁的模块结构，便于其他模块导入使用

### 2. `auto_updater.py` - 自更新管理器
- **功能**：完整的自更新流程管理
- **核心组件**：
  - **AutoUpdater类**：自更新管理器主类
  - **进度状态管理**：实时跟踪更新进度和状态
  - **文件下载器**：集成文件下载功能
  - **MD5校验**：确保文件完整性
  - **备份管理**：自动创建文件备份
  - **批处理执行**：调用外部批处理脚本完成更新

- **主要特性**：
  - 支持同步和异步两种调用方式
  - 完整的错误处理和状态跟踪
  - 与外部批处理脚本的无缝集成
  - 详细的日志记录和进度报告

## 更新流程
1. **版本检查** → 2. **文件下载** → 3. **完整性校验** → 4. **备份创建** → 5. **更新执行**

## 技术特点
- 基于异步编程模型，提高更新效率
- 集成MD5校验机制，确保文件安全
- 支持自动备份，防止更新失败导致数据丢失
- 与系统服务管理工具（NSSM）协同工作

## 使用场景
- 应用启动时的自动版本检查
- 手动触发的强制更新
- 后台静默更新（需管理员权限）

## 依赖关系
- 依赖于 `utils` 模块的文件操作、网络通信和路径处理功能
- 与 `scripts/automatic_update.bat` 批处理脚本协同工作
- 需要系统管理员权限执行更新操作
